A simple plugin that ensures images are included within your WordPress RSS feeds.

== Description ==

Img To RSS is a simple, no-frills plugin that automatically adds images to your WordPress RSS feeds. With this plugin, each entry in your RSS feeds will have an image field populated with the featured image of your posts, making sure that your content is engaging and shares well on platforms that utilize RSS feeds.

More info: [NoFrillsPlugins Img To RSS](https://www.nofrillsplugins.com/imgtorss)

== Installation ==

1. Upload the `Img To RSS` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it! The plugin will automatically start adding images to your RSS feeds.

== Frequently Asked Questions ==

= Does this plugin require any configuration? =

No configuration needed. Activate the plugin, and it will automatically add image fields to your RSS feed.

= What kind of images does the plugin add to the RSS feed? =

The plugin adds the featured image of your posts to the RSS feed as an <enclosure> element, which is the standard way of including media in RSS feeds.
